## Running Examples

All examples are available in the [webreel repository](https://github.com/anomalyco/webreel/tree/main/examples). Clone the repo and run any example:

```bash
git clone https://github.com/anomalyco/webreel.git
cd webreel/examples/hello-world
webreel record
```
